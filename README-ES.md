* Separa el front-end (capa de presentación) del back-end (gestión de contenido).
* Entrega contenido a través de APIs.

## 📥 Instalación

### Inicio rápido con Docker Compose

Clona el repositorio y ejecuta Nodify con todas sus dependencias (MongoDB y Redis incluidos):

```bash
git clone https://github.com/AZIRARM/nodify.git
cd nodify
docker compose up -d
```

Nodify estará disponible en:
* **Nodify Studio:** http://localhost:7821
* **Nodify API:** http://localhost:7805
* **Nodify Core:** http://localhost:7804

### Configuración de Docker Compose

```yaml
services:
  mongodb:
    image: mongo:latest
    container_name: nodify-mongodb
    volumes:
      - mongodb_data:/data/db
    ports:
      - "27017:27017"
    restart: unless-stopped

  redis:
    image: redis:latest
    container_name: nodify-redis
    volumes:
      - redis_data:/data
    ports:
      - "6379:6379"
    restart: unless-stopped

  nodify-core:
    image: azirar/nodify-core:latest
    container_name: nodify-core
    environment:
      MONGO_URL: "mongodb://mongodb:27017/nodify"
      ADMIN_PWD: Admin13579++
      API_URL: "http://nodify-api:1080"
      TZ: "${TZ:-Europe/Paris}"
      REDIS_URL: "redis://redis:6379"
      JAVA_OPTS: "-Xmx768m -Xms384m"
    ports:
      - "7804:8080"
    depends_on:
      - mongodb
      - redis
    restart: unless-stopped

  nodify-api:
    image: azirar/nodify-api:latest
    container_name: nodify-api
    environment:
      MONGO_URL: "mongodb://mongodb:27017/nodify"
      TZ: "${TZ:-Europe/Paris}"
      REDIS_URL: "redis://redis:6379"
      JAVA_OPTS: "-Xmx512m -Xms256m"
    ports:
      - "7805:1080"
    depends_on:
      - mongodb
      - redis
    restart: unless-stopped

  nodify-studio:
    image: azirar/nodify-studio:latest
    container_name: nodify-studio
    ports:
      - "7821:80"
    environment:
      CORE_URL: "http://nodify-core:8080"
      API_URL: "http://nodify-api:1080"
    depends_on:
      - nodify-core
      - nodify-api
    restart: unless-stopped

volumes:
  mongodb_data:
  redis_data:
```

### Variables de Entorno

| Variable | Descripción | Predeterminado |
| :--- | :--- | :--- |
| `MONGO_URL` | Cadena de conexión a MongoDB | `mongodb://mongodb:27017/nodify` |
| `REDIS_URL` | URL de conexión a Redis | `redis://redis:6379` |
| `ADMIN_PWD` | Contraseña del administrador | `Admin13579++` |
| `API_URL` | URL pública de la API | `http://nodify-api:1080` |
| `CORE_URL` | URL del servicio Core | `http://nodify-core:8080` |
| `JAVA_OPTS` | Opciones de JVM para Core y API | Ver arriba |
