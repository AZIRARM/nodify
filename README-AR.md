# 🚀 Nodify: أقوى نظام إدارة محتوى (Headless CMS) مفتوح المصدر كخدمة محتوى حقيقية (CaaS)

**تقديم تجارب محتوى سلسة عبر جميع القنوات — يدعم اللغات المتعددة بشكل أصلي، أي محتوى، لأي قناة.**

## فهم نظام Headless CMS

* يفصل بين الواجهة الأمامية (طبقة العرض) والخلفية (إدارة المحتوى).
* يقدم المحتوى عبر واجهات برمجة التطبيقات (APIs).
* قابل للتخصيص والتكيف بشكل كبير.

<img src="assets/pictures/headless-cms.png" width="50%"/>

## لماذا تختار Nodify؟

* **متعدد اللغات:** إنشاء وإدارة المحتوى بلغات متعددة بسهولة.
* **متعدد القنوات:** توصيل المحتوى إلى أي جهاز أو منصة.
* **قابل للتخصيص للغاية:** صمم نظام إدارة المحتوى حسب احتياجاتك الخاصة.
* **قابل للتوسع:** التعامل بسهولة مع أحجام المحتوى المتزايدة.
* **صديق للمطورين:** واجهات برمجة تطبيقات قوية وتكاملات سلسة.

## 📥 التثبيت

### البدء السريع باستخدام Docker Compose

قم بنسخ المستودع وتشغيل Nodify مع جميع التبعيات (بما في ذلك MongoDB و Redis):

```bash
git clone https://github.com/AZIRARM/nodify.git
cd nodify
docker compose up -d
```

سيكون Nodify متاحاً على الروابط التالية:
* **Nodify Studio:** http://localhost:7821
* **Nodify API:** http://localhost:7805
* **Nodify Core:** http://localhost:7804

### إعداد Docker Compose

```yaml
services:
  mongodb:
    image: mongo:latest
    container_name: nodify-mongodb
    volumes:
      - mongodb_data:/data/db
    ports:
      - "27017:27017"
    restart: unless-stopped

  redis:
    image: redis:latest
    container_name: nodify-redis
    volumes:
      - redis_data:/data
    ports:
      - "6379:6379"
    restart: unless-stopped

  nodify-core:
    image: azirar/nodify-core:latest
    container_name: nodify-core
    environment:
      MONGO_URL: "mongodb://mongodb:27017/nodify"
      ADMIN_PWD: Admin13579++
      API_URL: "http://nodify-api:1080"
      TZ: "${TZ:-Europe/Paris}"
      REDIS_URL: "redis://redis:6379"
      JAVA_OPTS: "-Xmx768m -Xms384m"
    ports:
      - "7804:8080"
    depends_on:
      - mongodb
      - redis
    restart: unless-stopped

  nodify-api:
    image: azirar/nodify-api:latest
    container_name: nodify-api
    environment:
      MONGO_URL: "mongodb://mongodb:27017/nodify"
      TZ: "${TZ:-Europe/Paris}"
      REDIS_URL: "redis://redis:6379"
      JAVA_OPTS: "-Xmx512m -Xms256m"
    ports:
      - "7805:1080"
    depends_on:
      - mongodb
      - redis
    restart: unless-stopped

  nodify-studio:
    image: azirar/nodify-studio:latest
    container_name: nodify-studio
    ports:
      - "7821:80"
    environment:
      CORE_URL: "http://nodify-core:8080"
      API_URL: "http://nodify-api:1080"
    depends_on:
      - nodify-core
      - nodify-api
    restart: unless-stopped

volumes:
  mongodb_data:
  redis_data:
```

### متغيرات البيئة (Environment Variables)

| المتغير | الوصف | الافتراضي |
| :--- | :--- | :--- |
| `MONGO_URL` | سلسلة الاتصال بـ MongoDB | `mongodb://mongodb:27017/nodify` |
| `REDIS_URL` | رابط الاتصال بـ Redis | `redis://redis:6379` |
| `ADMIN_PWD` | كلمة مرور المسؤول | `Admin13579++` |
| `API_URL` | رابط واجهة البرمجيات العام | `http://nodify-api:1080` |
| `CORE_URL` | رابط الخدمة الأساسية | `http://nodify-core:8080` |
| `JAVA_OPTS` | خيارات JVM لـ Core و API | كما موضح أعلاه |
