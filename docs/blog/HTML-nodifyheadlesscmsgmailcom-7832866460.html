Blog Template - Nodify

Welcome to the blog template for Nodify Headless CMS. Before using this template, please ensure you configure the necessary variables.

Important:

Please add a BASE_URL variable inside the values section of the parent node to ensure proper link generation.

Example:

  • If Nodify is deployed with Docker Compose: BASE_URL = /api
  • Otherwise, use the full Nodify API URL.

Once configured, you can start creating and managing your blog content efficiently!
