## 📥 安装指南

### 使用 Docker Compose 快速开始

克隆仓库并运行包含所有依赖项（MongoDB 和 Redis）的 Nodify：

```bash
git clone https://github.com/AZIRARM/nodify.git
cd nodify
docker compose up -d
```

Nodify 将运行在以下地址：
* **Nodify Studio:** http://localhost:7821
* **Nodify API:** http://localhost:7805
* **Nodify Core:** http://localhost:7804

### Docker Compose 配置参考

```yaml
services:
  mongodb:
    image: mongo:latest
    container_name: nodify-mongodb
    volumes:
      - mongodb_data:/data/db
    ports:
      - "27017:27017"
    restart: unless-stopped

  redis:
    image: redis:latest
    container_name: nodify-redis
    volumes:
      - redis_data:/data
    ports:
      - "6379:6379"
    restart: unless-stopped

  nodify-core:
    image: azirar/nodify-core:latest
    container_name: nodify-core
    environment:
      MONGO_URL: "mongodb://mongodb:27017/nodify"
      ADMIN_PWD: Admin13579++
      API_URL: "http://nodify-api:1080"
      TZ: "${TZ:-Europe/Paris}"
      REDIS_URL: "redis://redis:6379"
      JAVA_OPTS: "-Xmx768m -Xms384m"
    ports:
      - "7804:8080"
    depends_on:
      - mongodb
      - redis
    restart: unless-stopped

  nodify-api:
    image: azirar/nodify-api:latest
    container_name: nodify-api
    environment:
      MONGO_URL: "mongodb://mongodb:27017/nodify"
      TZ: "${TZ:-Europe/Paris}"
      REDIS_URL: "redis://redis:6379"
      JAVA_OPTS: "-Xmx512m -Xms256m"
    ports:
      - "7805:1080"
    depends_on:
      - mongodb
      - redis
    restart: unless-stopped

  nodify-studio:
    image: azirar/nodify-studio:latest
    container_name: nodify-studio
    ports:
      - "7821:80"
    environment:
      CORE_URL: "http://nodify-core:8080"
      API_URL: "http://nodify-api:1080"
    depends_on:
      - nodify-core
      - nodify-api
    restart: unless-stopped

volumes:
  mongodb_data:
  redis_data:
```

### 环境变量说明

| 变量名 | 描述 | 默认值 |
| :--- | :--- | :--- |
| `MONGO_URL` | MongoDB 连接字符串 | `mongodb://mongodb:27017/nodify` |
| `REDIS_URL` | Redis 连接 URL | `redis://redis:6379` |
| `ADMIN_PWD` | 管理员密码 | `Admin13579++` |
| `API_URL` | 公共 API 地址 | `http://nodify-api:1080` |
| `CORE_URL` | 核心服务地址 | `http://nodify-core:8080` |
| `JAVA_OPTS` | Core 和 API 的 JVM 选项 | 见上文配置 |
