# Installation Steps

1. **Clone the project from GitHub**  
   Open a terminal and run the following command:
   ```sh
   git clone https://github.com/AZIRARM/nodify.git  
   ```
2. **Navigate to the project directory**
   ```sh
    cd nodify
   ```

3. **Navigate to the project directory**
   ```sh
    docker compose up -d
   ```
